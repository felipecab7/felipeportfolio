Final deliverable

Augmented reality scene with the following requirements:

  • Animated movement or a special effect (either a scripted movement, animation using the animator window, custom particle effect, or collision in reality composer)

  • Custom lighting, or a custom image tag.

  • An audio component (looping background music, sound effect, ambient noise, etc)

  • Build Settings configured for AR on Android/Apple Mobile devices, Hololens, or iPad.

Image rendered using Unity


12 assets were modeled and scaled to real size using Maya, UVs were exported and painted using Substance Painter and the final scene was assembled in Unity along with shaders, candle animation and lighting.
