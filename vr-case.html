Creating a virtual reality movie set in Unreal






Overview
Modeled a 3D environment using Maya and Unreal for a Virtual Reality application aimed at film students of an introductory class at the University of Michigan. Analyzed archival material, modeled, textured, and rendered 29+ props..

Analysis of archival material

In order to achieve a realistic representation of the the envioronment I used photos of the set and broke down the clip into multiple stills images to get the basic proportions of objects and characters. In addition, I researched the height of the three actors and established an initial blocking of the scene.

Photos of the real set of citizen kane


Proportions

Getting the proportions right: I wanted to elimintate guess work as much as possible so I analized the trajectory of every character based on the clip and their height

Left: sketch, right: Initial 3D Blocking in Maya


Performance

In order to build an optimized environment that would work in any VR plaftform I kept the poly count very low and assigned only 28 materials to 80 meshes.

Modular meshes: I used a modular approach where many of the models would be copied inside Unreal to save memory.

  • Number of meshes: 80 meshes in the Maya. Only imported 29 to Unreal.

  • Number of materials: 28 materials.

  • Number of polygons: 12742.

  • Texel density: I kept the texel density consistent accross all meshes

Initial blocking displaying consistent texel density accross props
